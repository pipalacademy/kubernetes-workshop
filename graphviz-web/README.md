# Graphviz-web: Web Interface to Graphviz

The graphviz-web is a web interface to [Graphviz][], the open source graph visualization software. 

[Graphviz]: https://graphviz.org/

# How it run

Compile the go binary:

    go build web.go

Run the go server:

    ./web

Visit the website at:
 <http://localhost:8080/>
